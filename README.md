# 🏙️ My City OS

An AI-powered Smart Infrastructure and Citizen Reporting Portal built for **SDG-9** targets. Citizens can report local issues in seconds, and the system uses real-time automation to streamline resolutions.

---

## 🚀 Features
* **Multi-Language Support:** Seamlessly switch between English 🇬🇧 and Hindi 🇮🇳.
* **Auto-Location Detection:** Instant GPS pinpointing for accurate issue mapping.
* **AI Priority Tagging:** Smart sorting for emergency cases.
* **Gamified Citizen Score:** Earn badges for reporting verified issues.

## 🛠️ Tech Stack
* **Frontend:** React.js, Tailwind CSS
* **Routing & State:** React Router DOM, React Context API
